# ☁️ Cloud-Assisted Data Center – Online Training Platform

## 📌 Project Overview
This project demonstrates a **cloud-assisted data center model** designed for a medium-sized online training institute.  
It supports:
- 📺 Online classes (recorded)
- 📝 Online examinations
- 💾 Student data storage

The system is implemented using **web-based cloud services**, simulating a real-world scalable infrastructure.

## 🧱 Architecture

The platform follows a **cloud-based architecture**:

| Layer        | Implementation |
|-------------|--------------|
| Compute     | GitHub Pages / Netlify |
| Storage     | Google Drive |
| Streaming   | YouTube |
| Application | Google Forms |

## 💾 Storage Solution
The system uses **Software Defined Storage (SDS)** concept:
- Abstracts storage from physical hardware
- Enables centralized and scalable storage
- Improves flexibility and management

## ⚡ Scalability & Availability

### Scalability
- Supports dynamic user load
